### Deskripsi
Buatlah program untuk menghapus nilai-nilai duplikat pada sebuah array 1 dimensi `A` dengan elemen integer.

`A` adalah array yang di-inisialisasi di dalam program (tidak perlu jadi scanner input).


### Contoh 1
Input
```
[1, 2, 3, 3, 4]
```
Output
```
[1, 2, 3, 4]
```

### Contoh 2
```
[1, 2, 2, 3, 3, 4]
```
Output
```
[1, 2, 3, 4]
```


### Contoh 3
```
[1, 2, 3, 4]
```
Output
```
[1, 2, 3, 4]
```
