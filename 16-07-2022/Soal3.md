### Deskripsi
Buatlah sebuah program untuk menghitung bilangan faktorial.

Rumus faktorial adalah sebagai berikut.

***n! = n x (n-1) x (n-2) x ... x 3 x 2 x 1***


### Contoh 1
Input
```
4
```
Output
```
24
```
Penjelasan: 4 x 3 x 2 x 1 = 24


### Contoh 2
Input
```
1
```
Output
```
1
```

### Contoh 3
Input
```
0
```
Output
```
1
```
Penjelasan: 0! = 1
