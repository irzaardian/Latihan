### Description
Given an integer `n`, format it as a Rupiah currency.

Integer `n` can be expected to be tens, hundreds, thousand, millons, and so on.

### Example 1
Input
```
1500
```
Output
```
Rp 1.500,00
```


### Example 2
Input
```
987
```
Output
```
Rp 987,00
```


### Example 3
Input
```
107893
```
Output
```
Rp 107.893,00
```


### Example 4
Input
```
2456700
```
Output
```
Rp 2.456.700,00
```
